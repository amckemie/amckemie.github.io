Recent Work

At MakerSquare I worked on a variety of projects, with a focus on backend and frontend, including building my own versions of Foundation and jQuery, and designing the game of War via a LinkedList. My three main projects, however, are shown below and include:

Walking with the Dead: A Single Player, Text-Based Game

Walking with the Dead uses a PostgreSQL database with 11 tables to set up the game & constantly store any player's game state. Certain records in join tables are created/deleted as a player interacts with the world, its items & characters in the game. User input is parsed by searching for specific words, sending the player & input to a specific command, checking certain attributes, & returning values based on these arguments. It's currently at nearly 90% test coverage

Tech Stack: Ruby, PostgreSQL, ActiveRecord, Bcrypt, Colorize, Asciiart

Tree of Life

A web app that provides a dynamic visualization of the tree of life. I coded the backend of the project by parsing 400,000 lines of a JSON tree structure (using several recursive and helper methods) into a relational database with over 35,000 records. I also utilized Sinatra to create an API that returns a species’ basic data, its immediate children species (based on parent ID), and certain Wikipedia information.

Tech Stack: Ruby, RSpec, ActiveRecord, PostgreSQL, Postman, Infovis, Javascript, Sinatra

Give A Penny, Take a Penny

A web app created by a team of 3 that aims to connect volunteers and non-profit organizations through volunteer opportunities. I constructed the Volunteer/Organization entities, their database methods, the sign in/up commands, the layouts for entity profile and listing pages, and migrated the database to ActiveRecord.

Tech Stack: Ruby, Rpec, PostgreSQL, ActiveRecord, Sinatra, JavaScript, Foundation, Sass
